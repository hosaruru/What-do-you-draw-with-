# What do you draw with?

## サイト概要
### サイトテーマ
自分が何を使ってイラストを描いたかを共有できるサイト


### テーマを選んだ理由
私はデジタルイラストを描くのが好きなのですが、デジタルイラストではさまざまな種類のペンを使って描くことができます。
普段他人のイラストを見ていて、なんのペンを使って描いているのかが非常に気になる、またSNSで交遊している友達も同じように思っていることから
このアプリケーションを作りたいと思いました。

### ターゲットユーザ
SNSにイラストを投稿しているユーザー

### 主な利用シーン
自分が絵を描くときに参考にする時
